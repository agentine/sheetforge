# Changelog

## 0.1.0 — 2026-03-16

Initial release.

- Read and write Excel `.xlsx` files with a pure-Python, zero-dependency library
- API compatible with openpyxl where practical
- Cell types: string, number, boolean, formula, date
- Shared strings, merged cells, freeze panes, auto-filter
- Column and row dimension support
- Multiple worksheet support
- Full type hints with `py.typed` marker
- Python 3.10+
